## Configuration

Проект использует файл конфигурации `config.json` в формате JSON. Если его нет, его можно создать через программу.

### Структура `config.json`

```json
{
  "files": [
    "path/to/document1.txt",
    "path/to/document2.txt"
  ],
  "queries": [
    "пример запроса 1",
    "пример запроса 2"
  ]
}
